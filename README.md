# NFIS fylr plugin

This plugin is used by the [NFIS fylr](https://nfis.gbv.de) and adds the following functionality:

* When creating a new item, new areas of type "findspot" are created and linked to the item.


## Installation

The latest version of this plugin can be found [here](https://github.com/gbv/fylr-plugin-nfis/releases/latest/download/NFIS.zip).
